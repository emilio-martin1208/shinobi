Copilot Widget Demo

This page hosts a self-contained "editing copilot" widget (bottom-right). Select any text on this page and open the widget — the selection will be offered as context. Note: this demo calls the Anthropic API directly from the browser using a key you provide, which is fine for local testing but must not be shipped to production — proxy the request through your own backend instead.
