What is Shinobi?

Shinobi turns one long-form video into multiple short-form, vertical, subtitled clips — automatically. Upload a video or paste a YouTube link, and Shinobi finds the best moments, cuts them into clips, removes dead air, reformats them for Shorts/Reels/TikTok, burns in animated subtitles, and writes ready-to-post titles and descriptions for each clip.

How it works

  1. Transcribe — the audio is transcribed with word-level timing.
  2. Find moments — an AI model scans the transcript and picks the strongest 45–60 second segments based on hook strength, information density, and how well they stand alone.
  3. Clip — those segments are cut out of the source video.
  4. Remove silence — dead air and pauses are trimmed from each clip.
  5. Reformat to vertical — each clip is reframed to a 9:16 aspect ratio, automatically keeping faces in frame where possible.
  6. Add subtitles — animated, word-by-word captions are burned into the video.
  7. Generate metadata — a title, description, tags, and a Reddit-friendly framing are generated for each clip.

You can watch progress live as each step runs, then preview, download, edit the generated copy, and post the finished clips.

Supported platforms

YouTube Twitter / X Reddit

Connect your accounts from the app page to post finished clips directly. One-click posting is coming soon — for now, clips and metadata can be downloaded and posted manually.

Supported input

  • Video file upload — .mp4, .mov, .avi, .mkv
  • YouTube URL

Output

For each clip you get:

  • A vertical (9:16), subtitled video file
  • A generated title, description, and tags
  • Reddit-specific title and post text
  • Bulk download and "copy all metadata" actions

Generated files are temporary and are automatically cleared an hour after processing finishes.
